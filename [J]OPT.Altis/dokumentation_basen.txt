Alle auf dem Boden befindliche Gegenstände, sind hier auf Höhe 0 und horizontal in waage 0 gesetzt.
Somit können hier, alle auf dem Boden befindlichen Gegenstände, immer von 0 ausgehend, der Höhe der entsprechenden Map angepasst werden.
Hierbei ist darauf zu achten, das die Editorverzeichnisse wie unten aufgezählt sind, mit dem schloss veriegelt werden.

wenn diese Verzeichnisse mit dem Schloss veriegelt werden, können die restlichen objekte, alle zusammen markiert werden und alle gleichzeitig
gemeinsam, entsprechend der höhe angepasst werden.

Danach sollten dann die Verzeichnisse ohne Schloss, mit dem Schloss veriegelt werden.
Die zuvor veriegelt waren wieder geöffnet werden.
So kann mann dann die restlichen auch unproblematisch an die entsprechend gewünschte höhe anpassen.
